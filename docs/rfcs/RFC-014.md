# RFC-014: Engineering State Schema and Service Specification (Refined)

## 1. Abstract
Proposal ini merinci spesifikasi skema data `ENGINEERING_STATE.yaml` dan perilaku bisnis `EngineeringStateService` versi Refined yang kohesif dan modular.

---

## 2. Refined YAML Schema
Berkas `ENGINEERING_STATE.yaml` disimpan di bawah direktori `engineering/` dengan struktur sub-state berikut:

```yaml
version: "1"

project:
  id: "uuid-project-1234"
  name: "FlowForge Platform"

workspace:
  version: "1.0.0"
  engineering_phase: "coding"

mission:
  current_mission: "PROJECT-002"
  completed_missions:
    - "PROJECT-000"
    - "PROJECT-001"
  mission_history:
    - "PROJECT-000"
    - "PROJECT-001"
    - "PROJECT-002"

provider:
  current_provider: "Claude"
  provider_history:
    - provider: "Gemini"
      mission: "PROJECT-000"
      started_at: "2026-07-10T08:00:00Z"
      finished_at: "2026-07-10T08:30:00Z"
    - provider: "Ollama"
      mission: "PROJECT-001"
      started_at: "2026-07-10T08:31:00Z"
      finished_at: "2026-07-10T09:00:00Z"

session:
  current_session_id: "session-abc-123"
  latest_session_id: "session-xyz-987"

knowledge_state:
  knowledge:
    - id: "KNOW-001"
      title: "Repository Discovery"
      reference_path: "engineering/architecture/discovery.md"
      category: "repository"
      extracted_at: "2026-07-10T08:00:00Z"

decision_state:
  decisions:
    - id: "DEC-001"
      title: "Use SQLite for metadata persistence"
      rationale: "Requires Zero-Config offline operation"
      mission: "PROJECT-000"
      artifact_reference: "docs/adr/ADR-001.md"
      decided_at: "2026-07-10T08:05:00Z"

timeline_state:
  event_log:
    - id: "EV-001"
      event_type: "Mission Started"
      description: "Started mission: PROJECT-002"
      timestamp: "2026-07-10T09:01:00Z"

open_questions:
  - id: "QST-001"
    title: "Support multiple concurrent active sessions?"
    description: "Assess concurrency and database locks in SqlAlchemy"
    mission: "PROJECT-002"
    status: "open"
    created_at: "2026-07-10T08:15:00Z"
    resolved_at: null

blockers:
  - id: "BLK-001"
    description: "Redis server credentials are not configured in local cluster"
    created_at: "2026-07-10T08:10:00Z"

recommendations:
  - id: "REC-001"
    suggestion: "Initialize Distributed Worker test suite"
    proposed_at: "2026-07-10T08:15:00Z"
```

## 3. build_resume_context() API Output
Method `build_resume_context` merangkum status rekayasa saat ini menjadi format JSON netral:
```json
{
  "project_name": "FlowForge Platform",
  "engineering_phase": "coding",
  "current_mission": "PROJECT-002",
  "completed_missions": ["PROJECT-000", "PROJECT-001"],
  "active_blockers": ["Redis server credentials are not configured in local cluster"],
  "latest_decisions": [
    {
      "id": "DEC-001",
      "title": "Use SQLite for metadata persistence",
      "rationale": "Requires Zero-Config offline operation",
      "mission": "PROJECT-000",
      "artifact": "docs/adr/ADR-001.md"
    }
  ],
  "open_questions": [
    {
      "id": "QST-001",
      "title": "Support multiple concurrent active sessions?",
      "description": "Assess concurrency and database locks in SqlAlchemy",
      "status": "open"
    }
  ],
  "recommendations": ["Initialize Distributed Worker test suite"],
  "engineering_knowledge": [
    {
      "id": "KNOW-001",
      "title": "Repository Discovery",
      "category": "repository",
      "path": "engineering/architecture/discovery.md"
    }
  ]
}
```
*Engineering Context* ini disajikan bagi AI provider baru guna melanjutkan siklus orkestrasi proyek tanpa riwayat chat LLM.
