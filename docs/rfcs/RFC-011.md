# RFC-011: Smart Project Bootstrap Specification

## 1. Abstract
Proposal ini merinci spesifikasi pendeteksian bahasa dan framework secara otonom, struktur meta-informasi `WORKSPACE.yaml`, dan inisialisasi awal misi otomatis (`PROJECT-000`).

## 2. Project Detection Rules & Signatures
Detektor FlowForge mengenali berkas berikut di root proyek:
- **Django**: `manage.py`
- **Laravel**: `composer.json` + `artisan`
- **SpringBoot**: `pom.xml` (Maven) atau `build.gradle` (Gradle)
- **React**: `package.json` yang memuat dependency `"react"`
- **Vue**: `package.json` yang memuat dependency `"vue"`
- **Node.js**: `package.json` generik

## 3. Metadata Yaml Specification

### 3.1 `engineering/WORKSPACE.yaml`
Menampung karakteristik repositori hasil deteksi:
```yaml
workspace_type: "React"
language: "JavaScript/TypeScript"
framework: "React"
package_manager: "npm/yarn/pnpm"
build_tool: "vite"
```

### 3.2 `engineering/PROJECT_STATE.yaml`
Menampung pelacakan status pengerjaan misi:
```yaml
active_missions:
  - "PROJECT-000"
completed_missions: []
```
Jika Misi `PROJECT-000` berhasil di-`start` menggunakan CLI, maka statusnya dipindahkan secara otomatis ke dalam `active_missions`.
